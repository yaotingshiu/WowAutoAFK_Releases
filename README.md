# WowAutoAFK

WowAutoAFK 是一款專為《魔獸世界》玩家設計的自動化輔助工具。
協助玩家處理自動登入、掛機防斷等基本功能，以及自動釣魚，
使玩家遊戲中更輕鬆地處理事務。

## 🚀 主要功能

*   **自動登入系統**：支援帳號密碼加密儲存，快速進入遊戲。
*   **智能掛機防斷**：內建多種掛機模式，模擬真人按鍵操作。
*   **視覺化自動釣魚**：利用圖像識別技術自動偵測浮標咬餌。
*   **快捷鍵整合**：自定義掛機與釣魚快捷鍵，支援快速開關。
*   **掛機不無聊**：掛機全程可在背景處理(不包含釣魚)，不影響你的其他作業。

---

## 🛠️ 操作指南

### 1. 初次設定
1. 開啟程式後，請於主介面「主程式路徑」欄位，選擇您的 `WoW.exe` 執行檔路徑。
2. 在「登入動作延遲」區塊，依據您的電腦載入速度調整延遲秒數（建議設為 10-15 秒以上以確保載入穩定）。

### 2. 掛機功能使用
1. 在「自動掛機」分頁設定動作模式。
2. 透過設定好的「掛機快捷鍵」（預設 `Ctrl + F2`）即可快速啟動或停止掛機。

### 3. 自動釣魚功能 (需開啟釣魚分頁)
1. 在釣魚分頁設定拋竿快捷鍵。
2. 點擊「截取浮標圖片」，將區域框選至釣魚浮標。(範例)<img width="83" height="84" alt="2026-07-17_162752" src="https://github.com/user-attachments/assets/75c26d5c-5c10-4055-8f77-8ffda55d092b" />
3. 使用「釣魚快捷鍵」（預設 `Ctrl + F8`）開啟釣魚功能。
4. 避免使用浮動較大的浮標，可能會導致功能誤判。
> **注意：為確保執行順暢，釣魚期間請避免操作鍵盤與滑鼠，以免干擾程式判定。**

---

## ⚙️ 快捷鍵自定義
您可以透過設定分頁的下拉選單，自由更改掛機與釣魚的觸發快捷鍵。
*   系統支援 `Ctrl + F1` 至 `Ctrl + F9`。
*   **自動互斥機制**：系統已內建防衝突保護，掛機與釣魚功能將無法同時啟動。
